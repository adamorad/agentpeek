1

Install via Homebrew

Installs the binary and places the LaunchAgent plist on your system.

brew install adamorad/tap/airlock
2

Load daemon

Register with launchd so Airlock starts automatically at login.

cp $(brew --prefix)/share/airlock/com.airlock.daemon.plist ~/Library/LaunchAgents/ && launchctl load ~/Library/LaunchAgents/com.airlock.daemon.plist
3

Configure agent

Add to ~/.claude/mcp_servers.json

{"airlock": {"transport": "http", "url": "http://127.0.0.1:27183"}}
